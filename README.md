# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/568f86ff-cb71-4ddb-b5e1-474b18a190e2

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy [.env.example](.env.example) to `.env.local` and set `GEMINI_API_KEY`, `AIRTABLE_API_KEY`, `AIRTABLE_BASE_ID`, and `IMGBB_API_KEY`
3. Run the app:
   `npm run dev`
